extract_2D_peaks {GCxGC} R Documentation

extract GCxGC 2d peaks from the mzpack raw data file

Description

Usage

extract_2D_peaks(raw,
    mz = NaN,
    mzdiff = "ppm:30");

Arguments

raw

-

mz

target mz value for extract XIC data. NA means extract
TIC data by default.

mzdiff

the mz tolerance error for match the intensity data for
extract XIC data if the mz is not
NA value.

Details

this function will extract the TIC data by default.

Authors

Value

this function returns data object of type D2Chromatogram.

clr value class

  • D2Chromatogram
