Help For MZKit Library

MZKit

Data toolkits for processing NMR, MALDI MSI, MALDI single cell, LC-MS and GC-MS raw data, chemoinformatics data analysis and data visualization. The framework library for processing the mass spectrum rawdata files.

MSImaging

MS-imaging rendering package for R# environment based on the mzkit library and ggplot api.

MZKit HPC

Build high performance computing mzkit algorithm library based on Darwinism HPC runtime.
